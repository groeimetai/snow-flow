ServiceNow APIs Used

snow_query_table
Universal query tool for any ServiceNow table with smart analytics. OMIT limit for ALL records with minimal fields.
Key Parameters: table (required), query, limit, include_content, fields, group_by
snow_query_incidents
Advanced incident querying with filtering and analysis capabilities
Parameters: query, limit, include_content, include_analysis, fields
snow_analyze_incident
Deep incident analysis with pattern recognition and automated resolution suggestions
Parameters: incident_id, include_similar, include_suggestions
snow_auto_resolve_incident
Automated incident resolution based on known patterns with dry-run mode
Parameters: incident_id, dry_run (default: true)
snow_cmdb_search
CMDB configuration item search with relationship mapping
Parameters: query, ci_type, limit, include_relationships
snow_operational_metrics
Operational metrics and analytics including trends and KPIs
Parameters: time_period, metrics
Claude Code Usage: // Universal query for analytics - omit limit for ALL records const allIncidents = await mcp__claude-flow__snow_query_table({ table: 'incident', include_content: false, // Count only for performance group_by: 'category' }); // Query with limit for display const recentIncidents = await mcp__claude-flow__snow_query_table({ table: 'incident', query: 'state!=6^state!=7', fields: ['number', 'short_description', 'priority'], limit: 50 });

ServiceNow Machine Learning MCP
AI/ML capabilities with TensorFlow.js neural networks and ServiceNow native ML
15+ Tools TensorFlow.js No License Required

Key Features

  • Real TensorFlow.js neural networks (not simulated)
  • Works without expensive ServiceNow ML licenses
  • Intelligent data selection and optimization
  • Hybrid approach combining native and custom ML
  • Streaming support for large datasets
ml_train_incident_classifier
Trains LSTM neural networks on historical incident data with intelligent selection (up to 5000 records)
Parameters: training_size, auto_maximize_data, epochs, intelligent_selection, streaming_mode
ml_classify_incident
Real-time incident classification using trained neural networks
Parameters: incident_number, short_description, description
ml_hybrid_recommendation
Intelligent hybrid approach combining ServiceNow native ML with TensorFlow.js
Parameters: task_type, servicenow_weight, tensorflow_weight
Machine Learning Usage: // Train incident classifier with optimized data const training = await mcp__claude-flow__ml_train_incident_classifier({ auto_maximize_data: true, intelligent_selection: true, epochs: 50, streaming_mode: true }); // Classify new incident const classification = await mcp__claude-flow__ml_classify_incident({ incident_number: 'INC0000123', short_description: 'Email server not responding' });
