## üéØ **CRITICAL USAGE GUIDELINES FOR CLAUDE CODE**

### **Widget Debugging - ALWAYS Use Local Sync:**
```javascript
// ‚úÖ CORRECT - Use for widget debugging
snow_pull_artifact({ 
  sys_id: 'widget_sys_id',
  table: 'sp_widget' 
});
// Now use Claude Code native search, multi-file edit, etc.

// ‚ùå WRONG - Don't use for widget debugging  
snow_query_table({ 
  table: 'sp_widget',
  query: 'sys_id=...',
  fields: ['template', 'script', 'client_script'] 
});
// This hits token limits and can't use native tools!
```

### **ES5 Requirement for Server Scripts:**
```javascript
// ‚ùå WRONG - Breaks ServiceNow Rhino engine
const data = [];
data.forEach(item => process(item));

// ‚úÖ CORRECT - Works in ServiceNow
var data = [];
for (var i = 0; i < data.length; i++) {
  process(data[i]);
}
```
